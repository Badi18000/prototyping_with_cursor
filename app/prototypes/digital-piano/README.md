# Digital Piano

A minimalist luxury digital piano prototype inspired by macOS design language.

## Features

- **2 octaves** (C3 to C5) — 15 white keys and 10 black keys
- **Web Audio API** — plays synthesized piano-like tones (no external audio files)
- **Click or tap** — works on desktop and touch devices
- **macOS-inspired styling** — frosted glass, SF Pro typography, subtle shadows, traffic light window controls

## Design

- Frosted glass (backdrop-filter) window
- Refined color palette: light grays, soft whites
- Smooth press animations on keys
- Responsive layout

## How to run

The prototype is accessible via the homepage, or directly at:
`/prototypes/digital-piano`

Run the dev server with `npm run dev` and navigate to the prototype.
