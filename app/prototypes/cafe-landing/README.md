# Café Landing Page

Prototype de landing page pour un café, prêt à présenter à un client.

## Contenu

- **Hero** : Titre, sous-titre, CTA "Réserver une table"
- **Nos spécialités** : Grille de 3 cartes (Café, Viennoiseries, Pâtisseries)
- **Nous trouver** : Adresse et horaires
- **CTA final** : Réserver ou nous appeler
- **Footer** : Mention légère

## Personnalisation

Pour adapter à un vrai client :

1. Modifier le nom du café dans `page.tsx` (hero)
2. Changer l'adresse et les horaires
3. Remplacer le numéro de téléphone
4. Ajouter des images dans le dossier `images/` si besoin

## Lancer le projet

Le prototype est accessible via la page d'accueil du projet, ou directement à :
`/prototypes/cafe-landing`
